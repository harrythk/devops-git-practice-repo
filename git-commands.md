# Setup & Config Commands

git init >> Initialize Git repo

git clone <repo-url> >> Cone a remote repo

git remote -v >> View remote repo added in local config

git checkout -b <branch-name> >> to create a new branch


# Basic Workflow

git add <file_name> >> Stage a file

git commit -m "<message>" >> Commit staged files with message

git push origin main >> Push changes to remote

git pull origin main >> Pull changes from github remote repo

git branch >> To see the branch

git switch main  >> to switch branch


# Viewing Changes

git log --oneline >> To see the details of commits

git status >> Status of git stagingi
